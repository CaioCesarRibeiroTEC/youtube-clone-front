# 📺 YouTube Clone - Full Stack Challenge

![Banner ou Print do Projeto](link-da-imagem-aqui)

Este é um projeto Full Stack de alta fidelidade que recria a interface e funcionalidades core do YouTube. O objetivo foi aplicar conceitos avançados de arquitetura de componentes, gerenciamento de estado e integração com banco de dados.

## 🛠️ Tecnologias e Ferramentas

- **Frontend:** React, TypeScript, Styled Components, Vite.
- **Backend:** Node.js, Express.js.
- **Banco de Dados:** MySQL.
- **Context API:** Para gerenciamento de estado global (Sidebar, Usuário).

## 🚀 Funcionalidades Backend
- [x] API RESTful para gerenciamento de vídeos.
- [x] Modelagem de dados eficiente com MySQL.
- [x] Middleware de autenticação e rotas protegidas.

## 🎨 Funcionalidades Frontend
- [x] Layout 100% Responsivo.
- [x] Sidebar dinâmica com Context API.
- [x] Navegação SPA com React Router.

## 🔧 Como executar
1. Clone o repositório.
2. No diretório `/back`, execute `npm install` e `npm start`.
3. No diretório `/front`, execute `npm install` e `npm run dev`.
